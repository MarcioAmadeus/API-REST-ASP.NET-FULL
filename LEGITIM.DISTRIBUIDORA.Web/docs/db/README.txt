Para executar no Mac:

java -jar schemaSpy.jar -t sqlite2 -u admin -db ./db.sqlite -dp sqlite.jar -o output -gv "/usr/local/Cellar/graphviz/2.38.0"

Para executar no Windows:

java -jar schemaSpy.jar -t sqlite2 -u admin -db ..\..\App_Data\BD_BSPJ.s3db -dp sqlite.jar -o output -gv "C:\Program Files (x86)\Graphviz2.38"
